### Travel Options

### Booking System

- Each booking tracks:

  - Total price
  - Booking date
  - Status (Confirmed / Cancelled)
- **View your bookings**: Keep track of current and past bookings.
- **Cancel bookings**: Flexible cancellations directly from your dashboard.
